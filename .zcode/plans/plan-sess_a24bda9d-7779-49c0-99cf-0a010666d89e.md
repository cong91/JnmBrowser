# Kế hoạch: auto fallback location Nord khi ChatGPT chặn quốc gia

## Bối cảnh đã xác nhận
- Config đang dùng: `Nord · Vietnam #46` (`vpn_id=480cb3a1-...`).
- Worker VPN **có thể chạy được**, nhưng egress thực tế qua SOCKS là `45.80.187.66` → geo **HK**.
- Toàn bộ peer Nord gắn nhãn `VN` hiện tại đều geo thành **HK** (PacketHub), nên chỉ xoay peer trong VN **không hết lỗi**.
- ChatGPT qua tunnel này trả `403` / `unsupported_country_region_territory`.
- Login/Register hiện chỉ rotate peer với `country_id: None` và **không** phân loại lỗi quốc gia.

## Mục tiêu
Khi login/register gặp lỗi quốc gia không hỗ trợ:
1. Không retry mù cùng location.
2. Chuyển sang location Nord khác theo allowlist.
3. Restart vpn-worker, rồi retry attempt.
4. Áp dụng cho **cả Login + Register**.

Allowlist (bỏ country hiện tại):
`JP → SG → TW → KR → US → CA → GB → DE → NL → FR → AU`

Giới hạn an toàn: tối đa **3** lần đổi location / 1 credential (hoặc 1 alias CDK), rồi fail có message rõ.

## Thiết kế tối thiểu

### 1) Helper pure (test-first)
Thêm trong `src-tauri/src/vpn/nord.rs` (hoặc module nhỏ `auto_service/common/vpn_fallback.rs` nếu muốn tách khỏi Nord API):

- `is_unsupported_region_error(msg: &str) -> bool`
  - match case-insensitive: `unsupported_country_region_territory`, `request_forbidden` + country/region/territory, “Country, region, or territory not supported”.
- `const NORD_FALLBACK_COUNTRY_CODES: &[&str] = ["JP","SG","TW","KR","US","CA","GB","DE","NL","FR","AU"]`
- `next_fallback_country_code(current: Option<&str>, tried: &[String]) -> Option<&'static str>`
  - bỏ current + các code đã thử.
- `resolve_country_id_by_code(countries, code) -> Option<u32>`

### 2) Retarget location dùng lại private key
Thêm API dùng chung (ưu tiên trong `vpn/nord.rs` + thin wrapper storage):

```text
retarget_nord_vpn_to_country(vpn_id, country_code) -> Result<(server, conf), String>
```

Luồng:
1. `load_config(vpn_id)`
2. Chỉ cho phép `source == Some("nord")` (hoặc conf parse được + private key Nord style); reject conf không phải Nord.
3. Extract private key hiện tại.
4. `list_nord_countries` → map code → id.
5. `list_nord_wireguard_servers(Some(country_id), Some(40))`
6. Pick lowest-load server (tránh station/public key cũ nếu có).
7. `build_nord_wireguard_conf` + validate.
8. `update_config_data(vpn_id, conf, Some(default_nord_vpn_name(server)))`
9. (Khuyến nghị nhẹ) persist metadata mới nếu chạm model:
   - `source: "nord"`
   - optional `preferred_country_code` / `current_country_code` với `#[serde(default)]` để không phá config cũ.
10. `stop_vpn_worker_by_vpn_id` → sleep 1s → `start_vpn_worker`.

**Không** bắt buộc UI/Tauri command mới cho phase này; fallback chạy trong engine.

### 3) Gắn vào Login
File: `src-tauri/src/auto_service/openai/login/engine.rs`

Trong vòng retry credential (`login_once` loop ~824–900):
- Khi `Err(e)` **hoặc** `Ok(result)` với `!result.success` và message chứa:
  - nếu `effective_vpn_id()` có + còn budget fallback:
    - gọi retarget sang next allowlist country
    - log: `Unsupported region detected; switching Nord location VN → JP`
    - **không** tốn attempt mù; hoặc tốn 1 attempt nhưng luôn đổi location trước retry tiếp
    - continue retry

Cập nhật `rotate_wireguard_peer` success-path:
- truyền `country_id` theo country hiện tại/preferred nếu đã persist, thay vì luôn `None` (tránh nhảy lung tung khi success-rotate).

### 4) Gắn vào Register
File: `src-tauri/src/auto_service/openai/register/engine.rs`

Tương tự trong retry alias (`run_once` loop ~1640–1788):
- detect unsupported region → retarget location → retry
- hard-stop chỉ khi hết allowlist / retarget fail liên tiếp
- success-path `rotate_wireguard_peer` cũng dùng preferred country nếu có

### 5) Tests (không framework mới)
Trong `vpn/nord.rs` tests:
1. `is_unsupported_region_error` table-driven (positive/negative).
2. `next_fallback_country_code` bỏ current + tried, hết list → None.
3. (nếu extract helper pick) prefer different country than avoided station.

Optional pure unit cho retarget selection nếu tách được logic không cần network.

**Không** viết integration live Nord trong phase này.
