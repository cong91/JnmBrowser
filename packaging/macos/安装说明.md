# JnmBrowser macOS 测试版安装说明

这是一个未签名、未公证的测试版安装包。

## 安装步骤

1. 打开 DMG。
2. 将 `JnmBrowser.app` 拖到 `Applications`。
3. 如果系统提示：

   > “JnmBrowser”已损坏，无法打开。你应该将它移到废纸篓。

   请双击 `fix-quarantine.command` 执行修复脚本。
4. 修复完成后，再次打开 `JnmBrowser.app`。

## 脚本做了什么

脚本会执行下面这条命令，移除 macOS 的下载隔离标记：

```bash
sudo xattr -dr com.apple.quarantine "/Applications/JnmBrowser.app"
```

## 如果你把 App 装在了别的位置

可以在终端手动执行：

```bash
bash fix-quarantine.command "/你的/JnmBrowser.app"
```
